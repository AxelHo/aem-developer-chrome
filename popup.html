About

AEM Developer

This is a simple Chrome Extension written with AngularJS. It provides common links that the AEM developer uses on a daily basis, does a little string manipulation to alter the browser's location, and makes a few AJAX calls to the JCR.

I'm a Senior AEM Engineer at Adobe. This is a little side project of mine and is not an Adobe project.

I'd love to hear feedback and suggestions.
